# What is Xeger ?

Xeger is a regular expression tester app.

## How to use

When you open Xeger, you will see two textfields.

<Image src={Xeger1} />

You can input regular expression to the top textfield, and you can input test string to the bottom textfield, then you can see the result of the test.

| <Image src={Xeger2} width={256} /> | <Image src={Xeger3} width={256} /> |
| :---: | :---: |
| <div className="text-lime-400">○</div> | <div className="text-rose-600">×</div> |

If the test string matches the regular expression, the string will be green. If not, red.
